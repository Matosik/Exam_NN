# Exam NN — Подготовка к экзамену по нейронным сетям

Репозиторий для подготовки к курсу **«Нейронные сети и глубокое обучение» (НСиГО)**.  
Включает интерактивный тренажёр-викторину, материалы лекций, транскрипции и архив прошлого семестра.

---

## Быстрый старт

### Вариант 1 — двойной клик (без установки чего-либо)

Откройте `app/index.html` — тренажёр запустится прямо в браузере.

### Вариант 2 — через локальный сервер

```bash
cd app
python serve.py
```

Откроется `http://localhost:8000/index.html`.

> Разница: без сервера данные грузятся из `data/questions.js`; с сервером — из `data/questions.json` через `fetch`. Результат одинаковый.

**Требования:** Python 3 (только для варианта 2).

---

## Что такое тренажёр

Веб-приложение для самопроверки. Как работает:

- выбираете темы и количество вопросов (1–254);
- отвечаете на карточки с 4 вариантами ответа;
- карточка переворачивается — видите правильный ответ и пояснение;
- система интервального повторения (алгоритм Лейтнера) показывает трудные вопросы чаще, выученные — реже;
- прогресс сохраняется в браузере.

**База:** 254 вопроса по 6 темам, сгенерированы по материалам курса.

| Тема | Вопросов |
|------|----------|
| Введение в ИНС | 39 |
| Алгоритм обратного распространения | 31 |
| Обучение ИНС и ГНС | 45 |
| Свёрточные сети (СНС) | 47 |
| Рекуррентные сети (РНС) | 45 |
| Трансформеры | 47 |

**Горячие клавиши:** `1`–`4` — выбор ответа, `→` / `Пробел` — следующий вопрос.

## Как добавить или изменить вопросы

1. Отредактируйте `app/data/questions.json`.

2. Пересоберите JS:

   ```bash
   python app/tools/build.py
   ```

   Скрипт проверит данные (4 варианта, корректный индекс правильного ответа, уникальные `id`) и обновит `data/questions.js`.
